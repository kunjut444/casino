# Казино Frontend

Frontend приложение для мини-казино на React + TypeScript + Vite.

## Установка

```bash
npm install
```

## Запуск

```bash
npm run dev
```

Приложение будет доступно по адресу http://localhost:3000

## Сборка для продакшена

```bash
npm run build
```

## Функциональность

- Регистрация и вход пользователей
- Три типа ставок:
  - Четное/Нечетное
  - Числовая рулетка (1-36)
  - Эмодзи рулетка
- Отображение баланса игрока
- История ставок
- Современный и отзывчивый UI

## Технологии

- React 18
- TypeScript
- Vite
- React Router
- Axios
- CSS3
